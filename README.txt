##################################################################
	MUCHAS GRACIAS POR INSTALAR CAVE beta3.4.26!!!
##################################################################


Antes de iniciar con la instalacion debes de tener en cuenta ciertas cosas.

	1. Esta es la version beta estable de CAVE, asi que no habran cambios radicales en tu experiencia con retroarch.
	2. CAVE esta pensado para hacer lo mas simple posible la experiencia de emulacion de videojuegos. Este ejecutable se encarga de instalar retroarch en caso de que no lo tengas, y de crear playlist de distindos juegos (de nintendo gameboy advance, 64 y ds por ahora) para que sea cuestion de presionar iniciar y listo.
	3. El archivo ejecutable iniciara la descarga automaticamente, puedes darle doble click o ejecutar el comando './CAVE3_installer'. En caso de que la descarga no inicie, asegurate de que el ejecutable tenga permisos x 'ls -l'. En caso de no tener el permiso puedes darcelo con el comando 'chmod +x CAVE3_installer' e intentar la descarga otra vez.
	4. Si no tienes retroarch ya instalado, se te pedira que ingreses tu contrasegna para activar sudo, esto solo se encargara de descargar retroarch y no alterara mas alla ninguno de tus archivos dentro de root.
	5. Retroarch iniciara durante unos 5 segundos durante la descarga luego se volvera ha cerrar. Esto es normal y garantiza que las configuraciones sean las deseadas.
	6. Una vez la descarga termine busca y ejecuta retroarch, al final del menu deben estar las playlists ya creadas. Escoge un juego, dale a iniciar y disfruta.
	7. En caso de encontrar un error o querer instalar mas emuladores y juegos, se recomienda ver la documentacion de retroarch.
	8. En caso de estar leyendo este README desde github, puedes clonar este repositorio y entrar a la carpeta CAVEtest-3. Desde ahi puedes ejecutar 'pyinstaller CAVE3_installer' para crear el ejecutable. En caso de presentar algun error al ejecutar este comando, consulte con la comunidad de su distribucion ya que la respuesta puede variar.


Esperamos de que esta informacion sea util para tu experiencia retro :3.
